**Summary**  
Given positions of Cat A `x`, Cat B `y`, and Mouse C `z` on a number line, determine which cat reaches the mouse first (equal speed).  
Return:
- `Cat A` if `|z - x| < |z - y|`
- `Cat B` if `|z - y| < |z - x|`
- `Mouse C` if distances are equal

**Examples**  
- `x=1, y=2, z=3` → `Cat B`  
- `x=1, y=3, z=2` → `Mouse C`
