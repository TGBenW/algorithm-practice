**Summary**  
Given an array of integers, return the sum of its elements.

**Constraints**  
- `0 < n ≤ 1000`  
- `-1000 ≤ ar[i] ≤ 1000`

**Examples**  
Input:
6
1 2 3 4 10 11

Output:
31
