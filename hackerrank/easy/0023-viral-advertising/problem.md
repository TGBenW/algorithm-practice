**Summary**  
Day 1: ad is shared with 5 people. Each day, `liked = floor(shared/2)`, add to cumulative, then `shared = liked * 3`.  
Return the cumulative likes after `n` days.

**Example**  
`n=3` → liked per day: `2, 3, 4` → cumulative `9`.
